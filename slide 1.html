üîµ What is PHP?

Definition

PHP = PHP: Hypertext Preprocessor

A server-side scripting language designed specifically for web development.

Real-Life Analogy: Restaurant

üóÇÔ∏è HTML/CSS = Menu and table layout (interface)

üë®‚Äçüç≥ PHP = Chef in the kitchen (processes logic, cooks)

üè™ MySQL = Storage room (stores data)

The customer (browser) only sees the dish (HTML), not the chef working (PHP).

Why Learn PHP?

  • Popular: 77% of websites use PHP (WordPress, Facebook)
  • Easy to Learn: Simple syntax, rich documentation
  • Free: Open source, no licensing costs
  • Integration: Works seamlessly with MySQL, HTML, CSS
  • Community: Large community for support

üíª Basic PHP Syntax

PHP Tags

PHP code is placed between special tags:

<?php // PHP code goes here ?>

‚ö†Ô∏è Important Notes:

  • PHP file must have .php extension
  • Opening tag: <?php
  • Closing tag: ?> (can be omitted if file contains only PHP)
  • Each statement ends with semicolon ;

Output Statements

<?php echo "Hello World!"; // Most common print "Hello World!"; // Less common ?>

üìä Comparison: echo vs print

Feature echo print
Parameters Multiple ‚úì Only 1
Return Value None Returns 1
Speed Faster ‚ö° Slower
Usage More common üëç Less used

Examples

<?php // echo with multiple parameters echo "Hello", " ", "World!"; // print with single parameter print "Hello World!"; // Recommendation: Use echo in most cases ?>

‚úì Recommendation: Use echo in most cases for faster performance and multiple parameters.

üí¨ Comments in PHP

Single-Line Comments

<?php // This is a single-line comment # This is also a comment (less common) echo "Hello"; // Comment at end of line ?>

Multi-Line Comments

<?php /* This is a multi-line comment Use for long explanations or temporarily hide code */ /** * PHPDoc comment * Used for functions and classes * @param string $name User name * @return string Greeting message */ ?>

‚úì Best Practice: Write comments for complex code to explain the logic and help other developers understand your code.

üîÄ Embedding PHP in HTML

PHP can be embedded directly into HTML files:

<!DOCTYPE html> <html> <head> <title>PHP in HTML</title> </head> <body> <h1>Hello!</h1> <!-- Method 1: Using echo --> <p><?php echo "Date: " . date("Y-m-d"); ?></p> <!-- Method 2: Shorthand (if enabled) --> <p>Time: <?= date("H:i:s") ?></p> <!-- Method 3: PHP block --> <?php $name = "Student"; echo "<p>Hello $name</p>"; ?> </body> </html>

Key Points:

  • PHP runs on server, client only sees HTML
  • Mix PHP and HTML freely
  • Save as .php file (not .html)

‚úÖ Best Practices & Common Errors

Best Practices

  • Always use <?php instead of short tag <?
  • End statements with ;
  • Write comments for complex code
  • Use meaningful file names: login.php, products.php
  • Indent code for readability

Common Errors

// ‚ùå Wrong: Missing semicolon echo "Hello" // ‚úÖ Correct: echo "Hello"; // ‚ùå Wrong: Incorrect quotes echo "He said "Hello""; // ‚úÖ Correct: Escape or use single quotes echo "He said \"Hello\""; echo 'He said "Hello"';

üîß Common Installation Errors & Solutions

Error 1: Apache Won't Start

Cause: Port 80 is occupied by another application

Solution: Change Apache port to 8080 in configuration file

Error 2: MySQL Won't Start

Cause: Port 3306 is occupied or MySQL already running

Solution: End mysqld process in Task Manager and restart

Error 3: Blank Page or PHP Not Displaying

Cause: File doesn't have .php extension or syntax error

Solution: Check file extension and enable error display in php.ini

Error 4: Cannot Connect to MySQL

Cause: MySQL not started or wrong connection info

Solution: Check MySQL is running; verify host: localhost, user: root, empty password
